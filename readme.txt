A plugin for grading LearnDash assignments with enhanced features.

== Description ==
This plugin extends LearnDash by providing advanced assignment grading features. It allows instructors to grade assignments directly from the WordPress admin area and provides a streamlined interface for managing submissions.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/learndash-assignment-grading` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
